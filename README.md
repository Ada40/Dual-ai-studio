## üöÄ Quick Start

### Prerequisites

- **Python 3.8 or higher**
- **Tkinter** (usually included with Python)
- **Ollama** (optional, for enhanced local AI capabilities)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ada40/Dual-ai-studio.git
   cd Dual-ai-studio
   ```

2. **Install required dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Optional: Install Ollama for local AI**
   ```bash
   # Visit https://ollama.ai for installation instructions
   # After installation, pull recommended models:
   ollama pull llama2
   ollama pull codellama
   ```

4. **Launch Dual AI Studio**
   ```bash
   python main.py
   ```
